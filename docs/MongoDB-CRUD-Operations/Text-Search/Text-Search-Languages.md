
### Text Search Languages（文本搜索语言）
文本索引和[$text](#)操作符可用于下列语言，并接受两个字母的ISO 639-1语言代码或语言名称的长形式

| 语言名称     | ISO 639-1(双字母代码) |
| :----------- | :-------------------- |
| `danish`     | `da`                  |
| `dutch`      | `nl`                  |
| `english`    | `en`                  |
| `finnish`    | `fi`                  |
| `french`     | `fr`                  |
| `german`     | `de`                  |
| `hungarian`  | `hu`                  |
| `italian`    | `it`                  |
| `norwegian`  | `nb`                  |
| `portuguese` | `pt`                  |
| `romanian`   | `ro`                  |
| `russian`    | `ru`                  |
| `spanish`    | `es`                  |
| `swedish`    | `sv`                  |
| `turkish`    | `tr`                  |

> **注意**
>
> 如果您将语言值指定为**“ none”**，则文本搜索将使用简单的标记化，其中没有停用词列表，也没有词干。<br />
